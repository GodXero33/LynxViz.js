Installation



Currently, LynxViz is not available as a module script. Instead, you can include it directly in your HTML file using a script tag. Follow these steps to set up LynxViz in your project:


1.
Download LynxViz:Obtain the lynxviz.js file from here.
2.
Include the Library:Add the following script tag in the <head> section of your HTML file to include the LynxViz library:
<script type="text/javascript" src="path/to/lynxviz.js"></script>
3.
Add Your Custom Scripts:Include your custom JavaScript files after the LynxViz library to implement your drawing and animation logic:
<script type="text/javascript" src="path/to/your-script.js"></script>


Example HTML Setup



Here’s an example of how to set up your HTML file:

<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
		<title>Your Project Title</title>
		<script type="text/javascript" src="path/to/lynxviz.js"></script>
		<script type="text/javascript" src="path/to/your-script.js"></script>
	</head>
	<body>
		<div id="canvas-container"></div>
	</body>
</html>

In this example:

Replace "path/to/lynxviz.js" with the actual path to the lynxviz.js file.

Replace "path/to/your-script.js" with the actual path to your custom script file.



By following these steps, you’ll have LynxViz set up and ready to use in your project. Now you can start creating dynamic and interactive 2D graphics!
