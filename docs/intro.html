LynxViz.js

Welcome to the LynxViz.js where you can get help to make amazing animation in your website. In here you can create an animation loop easily using JS and animate anything you want. The library also provides lack of mathematical functions that haven't in default JS Math libray and some geometry related functions that can be helpful when animating with simulations. There are much more things up ahead and you gonna see everything in this library through this documentry.



Why use this library?

The main reason that I've created this library is to make thing easier for myself. The thing is every library out there that we can animate something in JS is not just includes animation thing witch make our web page much heavier.
